## Features
- Creating Gang
- Deleting Gang
- Updating Gang
- Resetting Gang
- Markers
    - Boss marker  
    - Take vehicle marker (integrated vehicles, purchased vehicles)  
    - Return vehicle marker  
    - Helicopter marker  
    - Levelling marker  
    - Stash  
- Stash System
    - Ability for everyone to enter stash if they have password (e.g., police can access if they have password). This feature can be disabled.  
    - Increasing stash size and weight by levelling up mafia (works on all inventories).  
- Action Menu
    - Cuff  
    - Uncuff  
    - Put in vehicle  
    - Take out of vehicle  
- Dealership
- Commands
    - Creating Gang  
    - Deleting Gang  
    - Teleporting to base  
    - Finding base via GPS (for players, can be disabled)  
    - Updating Markers Positions  
    - Updating Integrated Vehicles  
- Customization
    - Adding as many markers as you want  
- Garage
    - Integrated vehicles by admin
    - Purchased in Dealership
- Impound
- Logs System
- Levelling System
- Security & Performance
    - Protected against cheats  
    - Optimized (0.00 - 0.02 ms)
